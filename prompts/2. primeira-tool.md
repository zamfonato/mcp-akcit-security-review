<examples>
Few-shot com **dados empíricos reais**. Este é o formato exato que a tool
deve produzir.

**Input da tool MCP:**
```json
{ "name": "lodash", "version": "4.17.11", "ecosystem": "npm" }
```

**Output esperado da tool MCP (normalizado):**
```json
{
  "package": { "name": "lodash", "version": "4.17.11", "ecosystem": "npm" },
  "vulnerabilities": [
    {
      "id": "GHSA-29mw-wpgm-hmr9",
      "cve": "CVE-2020-28500",
      "summary": "Regular Expression Denial of Service (ReDoS) in lodash",
      "severity": "MODERATE",
      "cvss_vector": "CVSS:3.1/AV:N/AC:L/PR:N/UI:N/S:U/C:N/I:N/A:H",
      "cwe_ids": ["CWE-1333", "CWE-400"],
      "fixed_in": "4.17.21",
      "published": "2022-01-06T20:30:46Z",
      "advisory_url": "https://nvd.nist.gov/vuln/detail/CVE-2020-28500"
    }
  ],
  "queried_at": "2026-04-24T17:40:00.000Z"
}
```

**Regras de normalização** (aplicar nesta ordem):
  - `cve`: primeiro item de `aliases` que case com `^CVE-`, ou `null`.
  - `severity`: `database_specific.severity` se existir, senão `"UNKNOWN"`.
  - `cvss_vector`: primeiro item de `severity[]` com `type: "CVSS_V3"` ou
    `"CVSS_V4"`, campo `score`. Se não houver, `null`.
  - `cwe_ids`: `database_specific.cwe_ids` ou `[]`.
  - `fixed_in`: de `affected[]` — encontre o objeto cujo
    `package.name` bate com o pacote consultado, pegue
    `ranges[].events[]` e retorne o valor de `fixed` do primeiro evento
    que o tenha. Se nenhum, `null`.
  - `published`: string ISO direto.
  - `advisory_url`: primeira `references[].url` cujo `type === "ADVISORY"`;
    se nenhuma, primeira `references[].url` disponível; se o array
    estiver vazio, `null`.

**Caso de pacote sem vulns** (OSV retorna `{}`):
```json
{
  "package": { "name": "pkg-sem-vulns", "version": "1.0.0", "ecosystem": "npm" },
  "vulnerabilities": [],
  "queried_at": "2026-04-24T17:40:00.000Z"
}
```

**Caso de erro de domínio** (OSV retorna 400 ou rede falha) — retornar
MCP tool result com `isError: true` e content textual em PT-BR, ex:
`"Falha ao consultar OSV.dev: query inválida (HTTP 400)."`
</examples>
