# mcp-akcit-security-review

Servidor MCP em TypeScript que audita dependencias Node de uma codebase, consulta bases publicas por vulnerabilidades conhecidas e gera um relatorio `SECURITY-REVIEW.md` com analise de exploitability contextual. Trabalho pratico do Modulo 4 da Especializacao em Engenharia de Software com IA Generativa (AKCIT / UFG, 2026).
